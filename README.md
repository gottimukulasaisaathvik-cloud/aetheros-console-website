## About

Aetheros Console is a full-featured web application designed for managing and monitoring space settlement operations, including:

- **Admin Dashboard** - Full station management and system controls
- **Personnel Management** - Employee schedules, tasks, and announcements
- **Settlement Monitoring** - Real-time data on station systems
- **Resource Management** - Track and manage settlement resources
